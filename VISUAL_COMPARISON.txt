                          IMPLEMENTATION NOTES

When building the manual entry form:
  1. Show ALL 31 fields to leadership users
  2. Show only 25 fields to standard users
  3. Financial fields should have $ formatting
  4. Return rates should validate between 0-1 (0-100%)
  5. Dates should auto-populate where appropriate
  6. Required fields: Product name, SKU, Top Issue(s)

When exporting to Excel:
  1. Check user role BEFORE export
  2. Filter columns based on role
  3. Maintain exact column order as specified
  4. Include "Comments" sheet in both versions
  5. Apply header formatting (bold, gray background)
  6. Enable Excel auto-filter
  7. Auto-adjust column widths
  8. Use correct filename based on version

Security considerations:
  1. Leadership version should require authentication
  2. Log all exports of Leadership version
  3. Prevent screen capture on financial columns (if possible)
  4. Include data classification watermark
  5. Set document permissions appropriately
